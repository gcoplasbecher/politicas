Bem-vindo à Central de Documentação. Aqui você encontra links para os documentos essenciais exigidos pela Meta, bem como explicações sobre o que cada um contém.

Documentos Disponíveis

Política de Privacidade

Descrição: Este documento detalha como o aplicativo coleta, usa, armazena e compartilha suas informações pessoais. Ele explica os tipos de dados coletados (dados do usuário, técnicos e de integração com Meta), como eles são utilizados (para processamento de solicitações e segurança), medidas de segurança e os direitos dos usuários sob a Lei Geral de Proteção de Dados (LGPD) do Brasil.

Termos de Serviço

Descrição: Este documento define as regras gerais de uso do aplicativo, incluindo definições de termos, restrições de uso (como proibições de spam e violações das diretrizes da Meta), responsabilidades, limitações de responsabilidade e informações sobre propriedade intelectual. Ele assegura o uso responsável e legítimo dos serviços.

Política de Exclusão de Dados - LGPD

Descrição: Este documento instrui os usuários sobre como solicitar a exclusão permanente de seus dados pessoais, em conformidade com a LGPD. Ele descreve os tipos de dados elegíveis, o processo passo a passo de solicitação, métodos de exclusão e direitos relacionados, como portabilidade de dados.

Como Usar Esta Documentação

  • Clique nos links acima para acessar diretamente cada política.
  • Esses documentos são necessários para integração com APIs da Meta (WhatsApp, Instagram e Graph API).
  • Para dúvidas ou solicitações relacionadas, entre em contato via suporte@app.com.
