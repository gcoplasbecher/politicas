# Política de Privacidade

## Última Atualização: 09/12/2025

Esta Política de Privacidade descreve como o aplicativo coleta, usa, armazena e compartilha suas informações pessoais quando você utiliza nossos serviços. Ao usar este aplicativo, você concorda com as práticas descritas neste documento. Leia atentamente para entender como protegemos seus dados.

Este documento está em conformidade com a Lei Geral de Proteção de Dados (LGPD) do Brasil e regulamentações internacionais aplicáveis.

## 1. Informações que Coletamos

### 1.1 Dados Fornecidos pelo Usuário
- **Dados de Contato**: Nome, endereço de e-mail, número de telefone (para notificações via WhatsApp e Instagram).
- **Dados de Autenticação**: Credenciais utilizadas para acessar contas vinculadas às APIs da Meta (WhatsApp, Instagram).
- **Dados de Perfil**: Informações públicas do seu perfil nas plataformas Meta, como nome de usuário, foto de perfil e bio.

### 1.2 Dados Coletados Automaticamente
- **Dados de Uso**: Logs de acesso, ações realizadas no aplicativo, tempo de sessão, erros de sistema.
- **Dados Técnicos**: Endereço IP, tipo de dispositivo, sistema operacional, navegador utilizado.
- **Dados de Integração com Meta**: Identificadores únicos (tokens, IDs de usuário) gerados pelas APIs da Meta para comunicação via Graph API, webhooks de WhatsApp e Instagram.

### 1.3 Dados de Terceiros
- Quando você conecta contas Meta, recebemos dados diretamente das APIs da Meta, incluindo mensagens, mídia e interações.

## 2. Como Usamos Suas Informações

Utilizamos suas informações para fornecer e melhorar nossos serviços. Os usos incluem:

- **Processamento de Solicitações**: Permitir comunicação via WhatsApp e Instagram, enviando/recebendo mensagens e conteúdo.
- **Personalização**: Adaptar experiências baseadas em suas preferências e histórico.
- **Segurança e.Validação**: Verificar tokens de verificação (como verify_token) para autenticar webhooks.
- **Analytics e Melhorias**: Analisar uso para otimizar o desempenho e corrigir bugs.
- **Compliance Legal**: Atender a obrigações sob LGPD, incluindo solicitações de acesso/correção.

### 2.1 Compartilhamento de Dados
Não vendemos seus dados pessoais. Compartilhamos apenas quando necessário:
- **Com Meta Platforms**: Para autenticação e envio/recepção via APIs (WhatsApp Business API, Graph API).
- **Provedores de Serviços**: Hospedagem (ex: AWS, DigitalOcean), monitoramento de segurança.
- **Autoridades Legais**: Em resposta a ordens judiciais ou para cumprir leis aplicáveis.
- **Consentimento Explícito**: Com sua aprovação prévia para integrações específicas.

## 3. Armazenamento e Segurança dos Dados

### 3.1 Armazenamento
- Dados são armazenados em servidores seguros, criptografados em trânsito (HTTPS/TLS) e repouso (AES-256).
- Retenção padrão: Dados mantidos apenas enquanto necessário para os serviços; tokens temporários expiram automaticamente.
- Por padrão, excluímos dados desnecessários após 30 dias de inatividade, conforme LGPD.

### 3.2 Medidas de Segurança
- Autenticação OAuth para acesso às APIs Meta.
- Criptografia end-to-end para mensagens sensíveis.
- Firewalls, monitoramento 24/7 e atualizações regulares de segurança.
- Não compartilhamos dados sem consentimento; usamos ferramentas como ngrok para testes seguros.

## 4. Seus Direitos sob a LGPD

Conforme a Lei Geral de Proteção de Dados (LGPD), você tem os seguintes direitos:
- **Acesso**: Solicitar uma cópia dos dados pessoais processados.
- **Correção**: Atualizar informações incorretas ou incompletas.
- **Anonimização/Eliminação**: Solicitar a exclusão definitiva de seus dados (consulte nossa política de exclusão em `/docs/user-data-deletion.md`).
- **Portabilidade**: Obter seus dados em formato estruturado.
- **Revogação de Consentimento**: Cancelar consentimentos dados anteriormente.

Para exercer esses direitos, entre em contato via [inserir e-mail ou suporte].

### 4.1 Processo de Exercício de Direitos
1. Envie solicitação para [suporte@app.com] com identificação (nome, e-mail associado).
2. Será respondido em até 15 dias, conforme LGPD.
3. Para exclusão, follow o guia em `/docs/user-data-deletion.md`.

## 5. Uso por Menores

Nossos serviços não são direcionados a menores de 18 anos. Não coletamos dados intencionalmente de crianças. Se identificado, excluímos imediatamente os dados.

## 6. Cookies e Rastreamento

- Utilizamos cookies essenciais para funcionalidade (ex: manutenção de sessão).
- Não usamos tracking para publicidade; apenas analytics necessários.
- Configure seu navegador para gerenciar preferências de cookies.

Podemos atualizar esta política periodicamente. Notificaremos por e-mail ou notificações no app. Uso continuado constitui aceitação das mudanças.
